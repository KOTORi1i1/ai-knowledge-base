🧩 Skills 推荐 ​

精选好用的 Claude Code 斜杠命令、MCP 服务器和 AI 效率工具。每个推荐都附带安装教程和使用示例，新手也能快速上手。

什么是 Skills？ ​

Skills（技能）是能增强 AI 工具能力的扩展和插件。就像给手机装 App 一样，给 Claude Code 装上 Skills，它就能做更多事情：

  • 🔌 连接外部服务：让 Claude 操作数据库、浏览器、文件系统
  • 🧠 扩展专业能力：代码审查、深度研究、自动测试
  • ⚡ 提升工作效率：一键完成复杂任务

⚡ Claude Code 必装 Skills ​

/review🟢 新手友好
自动审查代码，发现潜在 bug、安全问题和优化建议。支持 PR 评审和本地代码审查。
/review 然后选择要审查的文件或 PR 链接
查看详情 →
/compact🟢 新手友好
压缩长对话上下文，让 Claude 自动总结历史并继续工作。对话变慢时的救星。
/compact 直接输入即可，无需参数

/init🟢 新手友好
一键初始化项目，自动生成 CLAUDE.md 配置文件，让 Claude 理解你的项目结构。
/init 在项目根目录执行

🔌 热门 MCP 服务器 ​

GitHub MCP🟡 需配置
让 Claude 直接管理 GitHub 仓库：创建 Issue、提交 PR、搜索代码、查看 Actions 状态。
需要 GitHub Personal Access Token
安装教程 →
Brave Search MCP🟡 需配置
为 Claude 提供联网搜索能力，获取最新信息。比内置搜索更稳定、结果更丰富。
需要免费注册 Brave Search API Key
安装教程 →
Filesystem MCP🟢 新手友好
让 Claude 安全地读写本地文件系统。比内置工具更灵活，支持批量操作。
几乎零配置，指定允许访问的目录即可
安装教程 →

🛠️ 开发者必备工具 ​

Aider🟡 需配置
命令行 AI 结对编程工具。在终端里和 AI 一起写代码，支持自动 commit、编辑多种文件。
pip install aider-chat 然后配置 API Key
查看详情 →
Ollama🟢 新手友好
本地运行大语言模型，完全免费、完全离线。支持 Llama、Mistral、DeepSeek 等开源模型。
一键安装，ollama run deepseek-r1 即可
查看详情 →
Cursor🟢 新手友好
基于 VS Code 的 AI 原生编辑器，内置 Chat、Composer、Tab 自动补全。目前最流行的 AI IDE。
从 cursor.com 下载安装，支持导入 VS Code 配置
查看详情 →
