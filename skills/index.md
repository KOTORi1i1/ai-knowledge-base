## 什么是 Skills？

Skills 是能**增强 AI 工具能力**的扩展和插件。就像给手机装 App 一样，Skills 可以让 Claude Code 做更多事。

## 分类浏览

|------|------|
| [Claude Code Skills](./claude-code) | Claude Code 内置的斜杠命令扩展 |

| [AI 效率工具](./tools) | 开发者常用的 AI 辅助工具推荐 |
