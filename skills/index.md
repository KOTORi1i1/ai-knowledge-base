# 🧩 Skills 推荐

> 精选好用的 Claude Code 斜杠命令、MCP 服务器和 AI 效率工具。每个推荐都附带**安装教程**和**使用示例**，新手也能快速上手。

---

## 什么是 Skills？

Skills（技能）是能**增强 AI 工具能力**的扩展和插件。就像给手机装 App 一样，给 Claude Code 装上 Skills，它就能做更多事情：

- 🔌 **连接外部服务**：让 Claude 操作数据库、浏览器、文件系统
- 🧠 **扩展专业能力**：代码审查、深度研究、自动测试
- ⚡ **提升工作效率**：一键完成复杂任务
